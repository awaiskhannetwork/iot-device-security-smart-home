# Securing Internet of Things (IoT) Devices in Smart Home Environments

## Project Overview

Designed and implemented a security-focused IoT environment simulating a smart home network. The project focused on identifying security risks associated with connected IoT devices and implementing security controls to protect devices, communications, and network infrastructure from unauthorized access and cyber threats.

## Objectives

* Identify security risks within a smart home IoT environment
* Perform network discovery and device enumeration
* Assess exposed services and potential attack surfaces
* Implement firewall and network security controls
* Strengthen authentication and access protection mechanisms
* Improve overall security posture of IoT devices

## Tools & Technologies Used

* Ubuntu Server
* Oracle VirtualBox
* Nmap
* Wireshark
* UFW Firewall
* Network Security Controls
* Linux Administration
* Virtual Lab Environment

## What Was Done

* Configured a smart home IoT lab environment using virtualized systems
* Performed network scanning and device discovery using Nmap
* Identified active hosts, open ports, and running services
* Reviewed network exposure and potential security vulnerabilities
* Implemented firewall controls using UFW
* Monitored network communications and traffic behaviour
* Assessed authentication and access control configurations
* Documented security findings, implemented controls, and testing results

## Outcome

Successfully assessed and secured a simulated IoT smart home environment through network monitoring, device discovery, vulnerability identification, and implementation of security controls. The project improved device protection, reduced exposure to network-based attacks, and strengthened overall IoT security.
